Available data files

You can either grab a specific datafile or the latest one.

Technical note: The urls below will redirect to the file hosted on another server. Please use the URLs below in your code as they will always work rather than the url that you are redirected to. You do not need to be logged in for these urls to work.

Latest Files

The url always points to the latest file. Listed below are the details of the current latest files.

Recent Files
